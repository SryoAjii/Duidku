#Duidku

Ini adalah project catatan keuangan yang dibuat menggunakan android studio dengan bahasa java. Program ini memungkinkan anda untuk mencatat pemasukan dan pengeluaran anda selama sebulan. Anda dapat mencatat pengeluaran apa saja yang anda lakukan dan akan ditampilkan totalnya, hal yang sama juga dapat dilakukan pada bagian pemasukan.

Fitur:
1. Menyimpan catatan keuangan anda
2. Menampilkan daftar pemasukan dan juga pengeluaran
3. Menampilkan total pemasukan dan juga pengeluaran

Cara penggunaan:
1. jalankan aplikasi Duidku
2. pilih bagian pemasukan atau pengeluaran
3. klik tanda + untuk menambah catatan keuangan
4. Masukkan detail seperti keterangan, tanggal, dan nominal
5. catatan keuangan akan di tampilkan
6. anda dapat menghapus catatan keuangan dengan klik tombol sampah
